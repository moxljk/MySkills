# Question Types

Use these as question categories, not as a mandatory sequence.

Read this file after you already know what kind of clarification move you want to make. This file helps with phrasing, not diagnosis.

## Clarify Meaning

Use when the user is using broad or slippery terms.

Prompts:
- "What do you mean by that in this context?"
- "If you put that more plainly, what are you actually claiming or asking?"

## Surface Assumptions

Use when a conclusion seems to rely on something unstated.

Prompts:
- "What are you taking for granted here?"
- "What would need to be true for this line of thinking to hold?"

## Distinguish Evidence From Inference

Use when the user sounds certain but the basis is unclear.

Prompts:
- "What part of this do you know, and what part are you inferring?"
- "Which part is observation, and which part is interpretation?"

## Examine Implications

Use when the user has a position but has not traced its consequences.

Prompts:
- "If this is right, what follows from it?"
- "What changes if you take this view seriously?"

## Introduce Another Perspective

Use when the user is stuck inside one framing.

Prompts:
- "How might someone thoughtful who disagrees see this?"
- "What is another reasonable way to understand the same situation?"
