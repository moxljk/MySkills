# Confusion Patterns

Read this file only when the user's confusion has a recognizable shape and you want help choosing the next move.

This file helps you diagnose the situation. It is not a prompt bank.

## Undefined Question

Signs:
- The user talks around a topic without stating the real question.
- Several concerns are present, but none is primary.

Moves:
- Offer two or three candidate questions.
- Ask what would count as clarity.

## Early Intuition, Unclear Reason

Signs:
- The user senses that something matters or something is wrong, but cannot yet explain why.
- Their reaction is clearer than their reasoning.

Moves:
- Ask what part feels important, weak, or inconsistent.
- Ask whether the issue seems closer to logic, framing, evidence, or implications.
- Compare the current version to a nearby version that would feel less wrong.

## Mixed Layers

Signs:
- Facts, interpretations, emotions, values, and actions are blended together.

Moves:
- Separate observation from interpretation.
- Separate preference from prediction.
- Separate values from tactics.

## Large Task, Unclear Structure

Signs:
- The user has a large todo, project, or responsibility but cannot yet see the main workstreams clearly.
- They are asking where to start before the task has been broken into workable parts.

Moves:
- Separate the large task into a few concrete directions or workstreams.
- Identify which parts depend on others and which can start immediately.
- Suggest a practical starting point rather than trying to solve the entire task at once.
- Stop at a workable map of subproblems, dependencies, and first steps; do not slide into detailed implementation unless the user explicitly asks for that shift.

## Reasoning Leap

Signs:
- The user jumps from premise to conclusion too quickly.
- The chain feels persuasive but incomplete.

Moves:
- Ask for the missing middle step.
- Test the conclusion against an alternative explanation.

## Scope Too Large

Signs:
- The topic is too broad to make progress.
- The user asks for total clarity all at once.

Moves:
- Shrink to the next decision, claim, or paragraph.
- Clarify what can remain unresolved for now.

## Tentative Direction

Signs:
- The user wants some direction but resists being pushed into a final conclusion.
- Several possibilities still feel alive.

Moves:
- Name the leading direction without claiming finality.
- Keep one or two alternatives visible.
- Suggest the next discriminating question or lightweight test.
